Información Personal

Hola, soy Erick Ortiz Velásquez, Desarrollador de Software Full Stack con experiencia en la creación de aplicaciones web modernas y escalables. Combino conocimientos en Angular para el Front-End con Node.js en el Back-End, y manejo bases de datos SQL utilizando Sequelize y MySQL. Además, domino herramientas como GitHub para mantener un control de versiones eficiente, y sigo metodologías ágiles para asegurar el éxito en cada proyecto. Mi objetivo es desarrollar soluciones personalizadas que optimicen la experiencia del usuario y garanticen un rendimiento robusto.

Habilidades

HTML

HTML es el lenguaje de marcado que utilizo para definir la estructura y el contenido de las páginas web. Es la base sobre la cual se construyen todos los elementos visibles y organizacionales en un sitio web.

CSS

CSS es el lenguaje de estilos que utilizo para darle forma y diseño a las páginas web. Con CSS, aplico estilos visuales y layouts responsivos, creando interfaces de usuario atractivas y usables.

JavaScript

JavaScript es un lenguaje de programación que utilizo para añadir interactividad a las páginas web, permitiendo la creación de interfaces dinámicas y funcionalidades avanzadas.

TypeScript

TypeScript es un lenguaje de programación basado en JavaScript que utilizo para escribir código más robusto y con tipado estático, facilitando el desarrollo y mantenimiento de aplicaciones complejas.

Angular

Angular es un framework de desarrollo de aplicaciones web que utilizo para construir aplicaciones web escalables y de alto rendimiento, facilitando la gestión de la lógica y la interacción con los datos.

Node.js

Node.js es un entorno de ejecución para JavaScript que utilizo en el desarrollo de aplicaciones del lado del servidor, permitiendo crear aplicaciones web rápidas y escalables.

Sequelize

Sequelize es un ORM (Object-Relational Mapping) que utilizo en el lado del servidor para interactuar con bases de datos SQL de manera eficiente. Con Sequelize, manejo las operaciones CRUD (Crear, Leer, Actualizar, Eliminar) y defino modelos de datos robustos, asegurando una gestión ordenada y estructurada de la información.

MySQL

MySQL es un sistema de gestión de bases de datos relacionales que utilizo para almacenar y gestionar datos de forma eficiente, permitiendo consultas complejas y persistencia de datos en aplicaciones web.

GitHub

GitHub es una plataforma de control de versiones que utilizo para gestionar, compartir y colaborar en proyectos de desarrollo. Con GitHub, mantengo un historial claro de cambios en el código, facilito la colaboración en equipo y gestiono el ciclo de vida de las aplicaciones desde el desarrollo hasta la implementación.
